# lein-rum

A drunken version of lein-run

## Usage

Put `[lein-rum "0.1.0"]` into the `:plugins` vector of your
`:user` profile, or if you are on Leiningen 1.x do `lein plugin install
lein-rum 0.1.0`.

    $ lein rum -m main.class

For usage type `lein help rum`.
